# 🃏 Blackjack Game

A simple Blackjack game built with **HTML, CSS, and JavaScript**. This project simulates the basic rules of Blackjack, allowing players to draw cards and try to reach a total of **21** without going over.

## 🚀 Features

* Random card generation
* Player information display
* Draw new cards during gameplay
* Blackjack win detection
* Game-over detection when the player exceeds 21
* Dynamic updates using JavaScript and DOM manipulation

## 🎮 How to Play

1. Start the game.
2. Two random cards are dealt.
3. Try to reach a total of **21**.
4. Draw a new card if your total is below 21.
5. If your total becomes 21, you win.
6. If your total exceeds 21, you lose.
